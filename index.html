Mahasiswa Ilmu Komputer Fakultas MIPA


Saya merupakan mahasiswa Ilmu Komputer yang memiliki ambisi untuk mengeksplorasi dunia pengembangan web. Dengan minat dan kemampuan saya dalam pengembangan software dan teknologi web, saya terus belajar dan mengasah keterampilan saya untuk menjadi seorang web developer yang kompeten.

Saat ini, saya aktif sebagai pengurus dalam organisasi Badan Eksekutif Mahasiswa Universitas Lambung Mangkurat dan aktif dalam Unit Kegiatan Mahasiswa Mipa Sport Corner Fakultas Matematika dan Ilmu Pengetahuan Alam dan saat ini juga bekerja dalam dunia Fotografi yaitu sebagai Photographer.



Pengalaman saya terkait website melibatkan:


  • Pembelajaran dan praktik aktif dalam pemrograman web dasar
  • Penggunaan platform dan alat pengembangan seperti Visual Studio Code, Netbeans dan Google Colab
  • Keterlibatan dalam proyek-proyek kelompok untuk meningkatkan keterampilan kolaboratif.


Minat dan Ketertarikan


Saya tertarik untuk mendalami bidang-bidang seperti pengembangan website, aplikasi mobile dan kecerdasan buatan. Saya percaya bahwa teknologi dapat menciptakan perubahan positif, dan saya berkomitmen untuk berkontribusi dalam mewujudkannya.
